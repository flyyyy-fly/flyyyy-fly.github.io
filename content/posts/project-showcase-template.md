## 🚀 项目概述
> 一句话简介：这是一个为了展示个人技术实力并实现自动化发布的博客系统。

### 🛠️ 技术栈
| 类别 | 技术项 |
| :--- | :--- |
| 框架 | Hugo (Golang) |
| 主题 | PaperMod |
| 部署 | GitHub Actions |
| 版本控制 | Git |

### 🌟 核心成果
- **访问优化**：通过配置静态资源，实现了全球秒开。
- **自动化流**：配置了 CI/CD，实现代码提交即发布。
